# Product Overview

This is a stock portfolio analysis and notification system for Chinese A-share markets. The application:

- Tracks stock positions and calculates technical indicators (MA, RSI, MACD, Bollinger Bands)
- Fetches real-time market data via Wind API
- Generates AI-powered weekly investment analysis reports using LLM
- Sends automated reports via WeChat (ServerChan) notifications

The system is designed for personal portfolio management with scheduled weekly analysis on Fridays at 4 PM.
